# Part 5: Arrays

This folder contains programs demonstrating array concepts in C++.

## Topics Covered

### Basic Arrays
- One-Dimensional Array
- Two-Dimensional Array
- Array Input & Output
- Array Traversal

### Array Operations
- Insertion
- Deletion
- Update
- Reverse
- Copy

### Mathematical Operations
- Sum
- Average
- Maximum Element
- Minimum Element
- Second Largest Element

### Searching
- Linear Search
- Binary Search

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort

### Intermediate Problems
- Merge Two Arrays
- Frequency of Elements
- Duplicate Elements
- Remove Duplicates
- Left Rotation
- Right Rotation

### Functions
- Passing Arrays to Functions
- Passing 2D Arrays to Functions

### Advanced Concepts
- Dynamic Arrays
- Array of Objects

## Learning Outcome

After completing this section, you will be able to create, manipulate, search, sort, and process one-dimensional and two-dimensional arrays while understanding how arrays are used in real-world C++ programs.
